📚 PrepMate – AI-Powered LMS for Smarter Learning
PrepMate is an AI-driven Learning Management System (LMS) that streamlines study material creation for exams, job interviews, and coding preparation. With intelligent content generation and seamless course management, PrepMate empowers learners with structured, high-quality study materials in just a few clicks.

✨ Key Features
AI-Powered Study Material Generator – Create detailed notes, summaries, and quizzes instantly.
Smart Course Management – Organize, track, and structure courses effortlessly.
Personalized Learning Paths – Adaptive AI-driven recommendations.
Interactive Learning Tools – Auto-generated flashcards, quizzes, and assessments.
Modern UI/UX – Built with Next.js and Tailwind CSS for a smooth user experience.
🚀 Tech Stack
Frontend: Next.js, Tailwind CSS, ShadCN
Backend: Drizzle ORM, Neon PostgreSQL
CMS & APIs: Hygraph, GraphQL
AI Integration: Gemini API
Deployment: Vercel
